## 🚀 Useful Kernel Commands

### Check Current Kernel Version

```bash
uname -r
```

### View Complete Kernel Information

```bash
uname -a
```

### Display Distribution Information

```bash
cat /etc/os-release
```

### View Boot Messages

```bash
dmesg | less
```

### List Loaded Kernel Modules

```bash
lsmod
```

---

## 🧪 Verify Kernel Information

### CPU Information

```bash
lscpu
```

### Memory Information

```bash
free -h
```

### Loaded Modules

```bash
modinfo ext4
```
